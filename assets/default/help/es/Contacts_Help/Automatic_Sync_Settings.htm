Opciones de sincronizacion

La aplicación puede ser configurada para arrancar automaticamente y llevar a cabo la acción elegida (ej. Sincronizar contactos).
Marque la casilla 'Petición de sincronización online' para permitir que la sincronización de contactos se dispare desde el servidor.
Marque la casilla 'Ajustes de sincronización automática' para realizar una sincronización de contactos a intervalos regulares. Existen dos tipos de sincronización automática: Autosincronización por cambios, se realizará la sincronización cuando se detecte que el número de cambios realizados en la agenda de contactos del teléfono es el indicado en detectar cambios. Y sincronización periódica, se realizará la sincronización en el período de tiempo y hora indicado. El valor especificado en el campo 'Detectar cambios' define el mínimo número de contactos que deben cambiar en el dispositivo desde la última sincronización.
El modo campo de sincronización define como debe comportarse la aplicación en el caso de la sincronización automática, cuando esté usando una red local, roaming o wifi. Seleccione una de las siguientes opciones:
Automático: Si se selecciona, la acción escogida comenzará automáticamente sin pedir confirmación. Nota: La casilla 'Datos en segundo plano' debe estar marcada en Ajustes del dispositivo > Cuentas y sincronización.
Recordar: Si se selecciona, se recibirá una notificación cuando sea el momento de sincronizar. Cuando la reciba, haga click en la notificación para disparar la sincronización automática.
No sincronizar: La acción escogida sera ignorada.
