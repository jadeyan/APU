Sync Options

The application can be configured to start automatically and perform a requested action (e.g. sync contacts).
Check the 'Online Sync Requests' checkbox to allow a contact sync to be triggered from your online account.
Check the 'Auto sync Changes' checkbox to perform a contact sync at regular intervals (as specified by the value set in 'Detect Changes' i.e. the minimum number of contacts have changed on device since the last sync).
The sync mode fields define how the application should behave for automatic sync, when you are using home network, romaing or wiki. Select one of the following options:
Automatic: If selected, the requested action starts automatically without confirming. Note: The 'Background data' checkbox must be ticked under Device settings > Accounts & sync.
Remind me: If selected, a notification will be received for the requested action. Click the notification triggers the sync automatically.
Never Sync: The requested action will be ignored.
